## 关键工程决策

1. **草稿独立 localStorage key**(`cys-stift.drafts.v1`,与 `cys-stift.cards.v1` 分离):草稿变化不触发卡片重渲染;草稿失败不影响卡片。
2. **草稿不进 domain**:web-local UI 状态,非核心业务实体;Phase 8 Tauri 端走 Tauri fs。
3. **`Draft.payload: unknown`**:各自 cast(capture `{title, body}` / manual 完整表单状态);不污染 type。
4. **防抖 500ms + useDebouncedCallback hook**:unmount cleanup,不在每次按键写 localStorage。
5. **Escape 保留 / 提交清除**:误关保护 + 显式 `draftStore.clear`。
6. **空草稿自动 clear**:所有字段空时清除(避免 stale)。
7. **snapshot 引用稳定**(同 db-client 模式)+ restore 用 `[ready]` deps,避免覆盖用户输入。
8. **CreateCardForm 改造不破坏 Phase 3 多媒介**:只加 useEffect 草稿读写 + 防抖 upsert,不动表单结构。
9. **0 新依赖** + **domain / db 零改动**。
