## 关键工程决策

1. **复用 `CardService.moveToCanvas`** + **`CanvasPosition`**;不重写。
2. **`DEFAULT_CANVAS_ID` 从 features/canvas 引用**;单向依赖。
3. **z 计算**:`Math.max(...existing.map(c => c.canvasPosition?.z ?? 0)) + 1`。
4. **位置阶梯式排布**:`100 + (z % 5) * 40`。
5. **详情状态用 `service.get(id)` 更新**,避免 stale。
6. **inbox 隐藏 on-canvas 卡**:Phase 2 `listInbox` 真相(spec §6.11)。
7. **Canvas dblclick 路径不动**;新路径并存。
8. **0 新依赖** + **domain / db 零改动**。
