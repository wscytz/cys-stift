## 关键工程决策

1. **CustomEvent `cys-stift:open-capture`**:不引入 Zustand;单实例 CaptureHost 是 open 状态唯一持有者。
2. **CaptureSinkRegistry**:模块单例 `Map<string, CaptureSink>`;Phase 8 TauriCaptureSink `register('tauri', ...)`。
3. **`openKind` 状态**:CaptureHost 追踪谁打开,save 时用对应 source.kind。
4. **MenuCaptureSink 与 WebCaptureSink 对称**:都走 `service.fromCapture`。
5. **动态 import + register**:service 注入,unmount 时 unregister。
6. **0 新依赖** + **domain / db 零改动**。
