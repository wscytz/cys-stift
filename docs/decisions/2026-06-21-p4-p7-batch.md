## Review 修复 #1 · ThemeBoot 未挂载(用户报)

**症状**: 用户在 `/settings` 手动切换日/夜间模式无效,DOM `data-theme` 不刷新。

**根因**: `apps/web/src/app/layout.tsx` 顶部 import 了 `ThemeBoot`,但 JSX body 里**没挂**。`useThemeApplication` hook 没运行,所以 settingsStore 通知没监听,`data-theme` 永远停在 head inline script 设的初值。

**修法**:
```tsx
<I18nProvider>
  <AppMenu />
  {children}
  <AIProviderSync />
  <ThemeBoot />        // ← 加这一行
  <CaptureHost />
  ...
```

**验收**:
- `pnpm --filter web build` exit 0
- `pnpm --filter web exec vitest run` 162/162 不回归
- GUI 实测:settings 切换 → 立即切换 dark/light

**commit**: `6f5902a`

## Review 修复 #2 · canvas toolbar 矩形/椭圆按钮死按钮(用户报)

**症状**: 工具栏 3 (矩形 r) 和 4 (椭圆 o) 按钮"有按压动画但不变化、不变红、画不出形状"。其它 6 个工具都正常。

**根因**: tldraw 3.x 把 `rectangle` / `ellipse` / `triangle` / ... 合并成单个 `'geo'` 工具,具体几何形状通过 `editor.setStyleForNextShapes(GeoShapeGeoStyle, 'rectangle'|'ellipse')` 切换。原代码调 `editor.setCurrentTool('rectangle')` 触发 tldraw 内部 `Error: root - no child state exists with the id rectangle.`,但 onClick handler 没 try/catch → **静默失败**。按钮按压动画是 CSS `:active`,与 handler 无关。

**puppeteer 实测确认**:
```
'geo': { ok: true, current: 'geo' }   ← 正确 tool id
'rectangle': { ok: false, err: "no child state exists with id rectangle" }
'ellipse': { ok: false, err: "no child state exists with id ellipse" }
```

**修法**:
1. import `GeoShapeGeoStyle` from `@tldraw/tldraw`
2. 抽 `activate(id)` 函数:rectangle/ellipse 先 `setStyleForNextShapes(GeoShapeGeoStyle, X)` 再 `setCurrentTool('geo')`;其他工具走 `setCurrentTool(id)`
3. active 高亮:rectangle/ellipse 改用 `getStyleForNextShape(GeoShapeGeoStyle)` 比较 + `current === 'geo'`;其他照旧
4. CSS / 快捷键 / i18n 不动

**puppeteer 修后实测**:
```
点击 矩形 按钮 → editor.tool='geo', nextGeoStyle='rectangle'
拖鼠标 (200,200)→(500,400) → 创建 shape {type:'geo', props:{geo:'rectangle'}, w:304, h:200}
点击 椭圆 按钮 → editor.tool='geo', nextGeoStyle='ellipse'
拖鼠标 (700,200)→(1000,400) → 创建 shape {type:'geo', props:{geo:'ellipse'}, w:304, h:200}
```

**验收**:
- `pnpm --filter web build` exit 0
- `pnpm --filter web exec vitest run` 162/162 不回归
- puppeteer 头less 真画出 rectangle + ellipse shape

**commit**: `48dfaa7`
