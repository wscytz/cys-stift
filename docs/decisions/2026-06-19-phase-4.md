## 关键工程决策

1. **tldraw v3.15.6（非 v5）**：spec 写 v3；v5 peer 要 React ≥19.2.1（我们 19.0.0），v3 peer `^18.2.0 || ^19.0.0` 匹配 + spec 对齐 + 不动 React。锁 v3。
2. **动态 import 边界在 `tldraw-canvas.tsx`**：tldraw 加载时访问 `window`，静态导出预渲染期炸。`useEffect` 内 `import('./canvas-editor')` 保证只在浏览器 mount 后加载；~2.1MB chunk 懒载不污染其他路由。
3. **shape id = `shape:` + cardId**：tldraw 强制 `shape:` 前缀（`shape ID must start with "shape:"`）；`cardShapeIdOf`/`cardIdFromShapeId` 双向映射，shape 与卡往返一致。
4. **`mergeRemoteChanges` 避自激**：加载变更标 remote 源，`store.listen({source:'user'})` 只听用户拖动，不触发回写循环。
5. **`pointerEvents: none` on HTMLContainer**：否则 HTML 覆盖层吞 pointer、tldraw 拖不动卡片。Phase 4 卡无内部交互 → tldraw 接管选中/拖拽，开详情走 DOM dblclick。
6. **`hideUi`**：隐藏冗余 chrome（形状工具条/菜单），保留选中/拖拽/缩放手柄。
7. **editor handle 经 `onEditorReady` 提到 page**：Modal 在 page 层，save/archive/delete 后用 helper 同步 shape。
8. **archived/deleted 过滤在 `loadCardsIntoEditor`**：不动 domain 的 `listOnCanvas`（spec §1.4 归档卡从画布隐藏）。
