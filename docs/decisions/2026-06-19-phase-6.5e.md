## 关键工程决策

1. **两路 capture 入口同一接口**:inbox 表单 + Mini Input 快捷键都走 `WebCaptureSink.submit → service.fromCapture`(spec §7 依赖倒置)。
2. **`CaptureInput.links` 是 `string[]`**,转换 `input.links.map(l => l.url)`。
3. **不 await `submit`**:同步 in-memory,无失败可能。
4. **`service.create` 仍保留**(canvas dblclick 路径用)。
5. **0 新依赖** + **domain / db 零改动**。
