## 0. 目标

详情 Modal 编辑模式**完整暴露** links/codeSnippets/quotes 编辑器(Phase 3 MVP 只暴露 title + body,违反 spec §4.2 字段完整性)。提交后 `CardService.update` 白名单已含 3 个字段(已实现,Phase 3)。
