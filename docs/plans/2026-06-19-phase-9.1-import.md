## 0. 目标

导出的 JSON 可**导入恢复**。/settings 加 Import JSON 按钮 → 选文件 → 校验 → 写 localStorage → 提示刷新。

**核心承诺**:Export → 清 localStorage → Import 同一文件 → cards/media/settings 全恢复。

---

## 1. 范围

### ✅ 做
- `apps/web/src/lib/export-service.ts` 加 `importFromJson(json: string): {cards, mediaAssets, drafts?, settings?}`:
  - 校验 `version === 1`
  - 校验基本 shape(cards 数组等)
  - 写回 4 个 localStorage key
- `/settings` 加 Import 按钮 + `<input type=file accept=".json">`
- 导入后提示 "Imported N cards — reload to apply" + 自动 reload

- 合并策略(merge vs replace)→ MVP 覆盖
- 冲突解决(同 id)→ 覆盖
- 导入预览 → 留后
- 撤销 → 留后(建议先 Export 备份)
