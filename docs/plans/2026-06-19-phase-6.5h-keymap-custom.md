## 0. 目标

`/settings` 路由可改全局 capture 快捷键(modifier + key),CaptureHost 读 settings 而非硬编码 `Cmd+Shift+Space`。

---

## 1. 范围

### ✅ 本阶段做

#### 1.1 settings store
- 新建 `apps/web/src/lib/settings-store.ts`:
  - localStorage key `cys-stift.settings.v1`
  - 类型:`{ captureShortcut: { modKey: 'meta' | 'ctrl', shift: boolean, code: string } }`
  - 默认:`{ modKey: 'meta', shift: true, code: 'Space' }`(等效 Cmd+Shift+Space)
  - API:`get() / update(patch) / useSettings()`

#### 1.2 `/settings` 路由
- `apps/web/src/app/settings/page.tsx`:
  - 显示当前快捷键
  - 下拉/输入改 modKey / code
  - Save → settingsStore.update

#### 1.3 CaptureHost 读 settings
- `capture-host.tsx`:
  - `useSettings()` 拿 captureShortcut
  - keydown 监听用 settings 值匹配(不再硬编码)
  - 默认 fallback 兜底

### ❌ 本阶段不做

- 冲突检测(快捷键已被浏览器/系统占用)→ 留后
- 录制式快捷键捕获(按下即设)→ 留后(MVP 下拉选)
- canvas 快捷键(`+ - 0 1 g`)自定义 → 留后
- Tauri 端读 settings → Phase 8
